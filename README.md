# 🗿🧻✂️ Rock paper scissors BOT

This is a gaming bot with which you can play paper rock scissors

### Commands

| Command                                | Description          |
|----------------------------------------|----------------------|
| **info**                               | Show info about bot. |
| **play** <_rock_\|_paper_\|_scissors_> | Play with bot.       |
| **help**                               | Shoe commands list.  |
